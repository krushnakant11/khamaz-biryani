# Khamaz Biryani - Restaurant Website

A modern, production-ready Next.js restaurant website with online ordering, reservations, and admin dashboard.

## Features

✅ Modern Design (Black-Gold-Orange Theme)  
✅ Responsive Mobile-First Design  
✅ Framer Motion Animations  
✅ Menu System with Filters  
✅ Shopping Cart with Zustand  
✅ Reservation Booking Form  
✅ Admin Dashboard (Orders, Reservations, Analytics)  
✅ API Routes (Orders & Reservations)  
✅ SEO Optimized  
✅ Google Maps Integration  
✅ Floating WhatsApp & Phone Buttons  
✅ TypeScript Support  

## Quick Start

```bash
npm install
cp .env.example .env.local
npm run dev
```

Open http://localhost:3000

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand
- Lucide Icons

## Deployment

Deploy to Vercel:

```bash
npm install -g vercel
vercel
```
